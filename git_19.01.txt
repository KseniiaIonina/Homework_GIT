First level: Коротко дайте ответы на след. вопросы:
(1) Зачем предназначен файл .gitignore?

Файл, который отмечен для GIT как игнорируемый

(2) Что делает команда git remote rm origin?

Удаляет подключение к удаленному репозиторий


(3) В каких случаях происходит MERGE CONFLICT?
Если вы обьединяете две ветки и в них есть файлы, который редактировался и в той и и другой ветке 

(4) Как посмотреть отличия между ветками?
Git diff ветка1 ветка2

(5) Какой командой удалить локальную и удаленную ветки?

Git branch -D ветка1

(6) Как сделать файл неотслеживаемым (untracked)?

git rm --cached name.txt
